Watts The Problem – Power Outage Prediction (DSC 80)

Feb 2025 – Mar 2025

Built a RandomForestClassifier pipeline to classify outage climate using a cleaned U.S. dataset. Improved accuracy from 60.6% to 75.4% using hyperparameter tuning and cross-validation. Developed an interactive GitHub Pages dashboard to present results with EDA and hypothesis testing.

Monte Carlo Stock Price Simulator – Triton Quantitative Trading

Oct 2024 – Dec 2024

Developed a Streamlit-based GBM simulator with real-time data from Yahoo Finance. Allowed users to simulate stock price movements based on time horizon and volatility inputs, with dynamic matplotlib plots and statistical confidence intervals.

VentureNex – Multi-Language Coding Assistant

Sept 2024 – Oct 2024

Created an AI assistant using Gradio and Ollama to answer programming queries in multiple languages. Added context-aware responses and chat history features using Python and real-time APIs.

Spotify Sentiment-Based Music Recommendation

Jan 2025 – Present

Engineering a personalized music recommender using TF-IDF + BERT embeddings, YouTube comment scraping, and Spotify Web API. Built a hashmap-based similarity engine and an adaptive feedback loop to learn from user interaction and improve song matching.
